# Веб-приложение "Погода"
Простое веб-приложение, которое позволит вам быстро и легко узнать погоду.

![aboba](https://github.com/user-attachments/assets/ad4f75d3-eee4-4810-9c85-c9de697ca41b)
## Установка
1. Зарегистрируйтесь на [OpenWeatherMap](https://openweathermap.org/) и получите свой API-ключ
2. Клонируйте репозиторий в среду разработки
3. Установите зависимости:
```bash
pip install -r requirements.txt
```
5. Запустите файл `app.py`
6. Перейдите по адресу `http://127.0.0.1:5000`
7. Введите свой API-ключ
